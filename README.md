# Online Learning Platform Usage Analysis (EDA Project)

## Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on an online learning platform usage dataset to uncover insights about user behavior, engagement, and learning patterns. The analysis helps understand how different users interact with online learning platforms and what factors influence their learning experience.

---

## Objectives

* Analyze user demographics such as age, gender, and role
* Identify popular learning platforms and devices used
* Study user engagement through time spent on learning
* Evaluate course completion rates and satisfaction levels
* Discover relationships between user behavior and learning outcomes

---

## Dataset Description

The dataset contains information about users of online learning platforms, including:

* **Age** – User age group
* **Gender** – Male/Female/Other
* **User Role** – Student, Professional, Freelancer
* **Platform** – Learning platforms (Coursera, Udemy, etc.)
* **Device** – Mobile, Laptop, Tablet, Desktop
* **Hours_Per_Week** – Time spent on learning
* **Purpose** – Learning goal (Career Growth, Skill Upgrade, Academic)
* **Completion_Rate** – Percentage of course completion
* **Satisfaction_Score** – User satisfaction rating

---

## Tools & Technologies Used

* Python 
* Pandas

* Seaborn

---

## Key Insights

* Majority of users belong to the **20–40 age group**, including both students and professionals
* **Mobile devices** are the most preferred mode of learning
* Most users learn for **career growth and skill development**
* Average course completion rate is around **50%**, indicating drop-offs
* Higher engagement (more hours spent) leads to **better completion rates**
* User satisfaction is moderate, suggesting room for improvement

---

## Visualizations Included

* Age & Gender distribution charts
* Platform and device usage analysis
* Time spent vs completion rate
* Satisfaction score distribution
* Correlation heatmap

---

## Conclusion

The analysis shows that online learning platforms are widely used for career advancement. While engagement levels are moderate, improving course quality and user experience can significantly increase completion rates and satisfaction.
